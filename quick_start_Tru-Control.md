## Quick Start (TL;DR)

1. Install and verify `inetbox.py` on your Raspberry Pi

3. Copy `mqtt_bridge_complete.py` and/or `truma_ble_server.py` to `/usr/local/bin`
4. Make both scripts executable
5. Configure MQTT credentials (if remote access is required)
6. Enable Bluetooth and configure the BLE device name
7. Install and enable the systemd services
8. Reboot the Raspberry Pi
9. Install the **Tru-Control** app from the Apple App Store
10. Connect via BLE (local) or Internet (MQTT)
